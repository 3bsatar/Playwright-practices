A comprehensive UI and API automation testing project built with Playwright, TypeScript, and Node.js.
This repository showcases practical, real-world test scenarios with a clean and scalable structure, including fixtures, page objects, reusable utilities, and modern automation best practices.


## Tech Stack
- **Playwright**
- **TypeScript**
- **Node.js 18+**
- **Allure Reports**
- **Page Object Model (POM)**
- **npm / Node Package Manager**


 ## Running Tests
You can execute the tests using **Playwright** and **npm**:

```bash
# Run all tests
npx playwright test

# Run tests in headed mode
npx playwright test --headed

# Run a specific test file
npx playwright test pom

# Run a test by title / test name
npx playwright test -g "E2E"
```



notes:
  - "Ensure Node.js 18+, npm, and Allure CLI are installed and added to your system PATH."
  - "Install Playwright browsers using: 'npx playwright install' (Chrome, Firefox, and WebKit must be installed)."
